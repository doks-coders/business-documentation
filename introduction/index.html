# Boogie Backend

The Boogie Backend is the backbone of the Sharp Detail Backend. It contains all the logic required to make the Sharpdetail Application flow in the right direction. The Boogie Backend has been structured into layers that will each work hand in hand to abstract the processes in the application. Our Aim to to understand the flow of the application and how we can improve it.
