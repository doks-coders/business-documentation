# Initialise Application

To retrieve specific key-value objects, you can use the key query to specify the property you want to use for querying, and the select query to specify the values you want to filter. In this case, you want to retrieve the object(s) whose Nutritional deficiencies is equal to [2277,2040,1846].

#

# Request

  • HTTP Method: GET
  • accept : application/json
  • Content Type : application/json
  • URL: http://localhost:3000/end-drug-data/data-select?title=1-annual-number-of-deaths-by-cause&key=nutritional deficiencies&select=[2277,2040,1846]

# Query Parameters

Property Type Required Description
title any true This parameter is used to select the title of the spreadsheet
key any true Selects the Column
select any true Selects the Specific values in the Column

# Request example

curl -X GET \ 
 'http://localhost:3000/end-drug-data/data-select?title=1-annual-number-of-deaths-by-cause&key=Nutritional deficiencies&select=[2277,2040,1846]' \ 
-H 'accept : application/json' \ 
-H 'Content-Type : application/json' \
